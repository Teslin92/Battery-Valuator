# Battery Valuator - Frontend

Modern React/TypeScript frontend for battery material valuation.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Visit: http://localhost:8080

## Build

```bash
npm run build
```

## Environment Variables

Copy `.env.example` to `.env.local` and configure:

- `VITE_API_URL` - Backend API URL (default: https://web-production-e2d0.up.railway.app)

## Deployment

This frontend is configured to deploy to Vercel. See `vercel.json` in the root directory.
